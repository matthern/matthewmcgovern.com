GIT Cheat Sheet

Some basic commands for starting with git.

Matthew McGovern · July 3, 2023

Some basic commands for starting with git:

Initialize & Clone

# Initialize a new Git repository in your current directory
git init

# Clone (copy) a repository to your machine
git clone <repo>

Make Changes

# Check status of your files - recommended before/after making changes
git status

# Stage files for a commit
git add <filename>

# Add all changed files to the stage
git add .
git add -A

# Commit files with a descriptive message
git commit -m "commit message"

Branching

# List all local branches in current repo
git branch

# Create new branch
git branch <branchname>

# Change to another branch
git checkout <branchname>

# Merge changes to (usually master) main branch
git merge <branchname>

Remote Repositories

# Connect to remote repo
git remote add origin <repo url>

# Check which remote servers are connected
git remote -v

# Add or delete remote URLs
git remote set-url --add <name> <url>
git remote set-url --delete <name> <url>

# Push commits to remote repository
git push origin <branchname>

# Pull and merge changes from remote repository
git pull

# Download changes without integrating them
git fetch

# Combine branches by creating a new commit
git merge

# Move or combine commits onto a new base
git rebase

# Discard local changes by moving the branch pointer
git reset

# Undo changes by creating a new commit that reverses the effects
git revert

Example Workflow

# Add one file to repo from working machine
git add shopping_list.yml
git commit -m "new file"
git push origin main

.gitignore

The .gitignore file is used to exclude certain files and directories from being tracked by Git. It uses pattern matching rules to specify what should be ignored.

By adding filenames, extensions, or directory names to the .gitignore file, you can prevent them from being committed and pushed to the repository. This helps keep the repository clean and avoids including irrelevant or sensitive files in the version history.
