NSX Edge Commands

October 15, 2023 Matthew McGovern

Reference for some NSX edge commands:

Check Interfaces

show interface

Check Log

show log

TCP Dump / Packet Capture

Choose NIC seen in sh int and IP required with host_ prefix. Can use 'capture' to record to pcap file and 'no debug' to stop. Can scp off the edge, show and remove these captures.

# Display packets on interface
debug packet display interface vNic_1 host_10.0.0.1

# Display packets by port
debug packet display interface vNic_x port_xxx

# Capture packets to file
debug packet capture interface vNic_x host_destinationip

# Stop capture
no debug packet capture interface vNic_1

# Show captured files
debug show files

# Copy files via SCP
debug copy scp

# Remove all debug files
debug remove all
