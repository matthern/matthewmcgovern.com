Interests

I love to use Linux and explore the command line, finding simple solutions with enough visibility to gain an understanding of the complete process. My interests include Linux system administration, infrastructure automation and experimenting with Proxmox and virtualization and Docker containers, monitoring, and scripting on my homelab.

My homelab began as an old laptop with Ubuntu, where I learnt the Linux filesystem and CLI. Since then it has expanded to a HA clustered Proxmox host setup with shared storage running a number of Linux VMs.

Along with a managed switch and a consumer grade NAS, I host my own dockerised applications and websites as a way to learn more about Python, Go and Bash scripting.

So far with my lab I have played with backup systems and scripts, Rancher, Wireguard VPN, DNS servers, file servers, NFS shared storage, media servers and USB/Video device passthroughs, reverse proxy and SSL certificates with Grafana, Telegraf/InfluxDB and Prometheus monitoring using custom dashboards for my UPS and NAS.

Outside of IT, I love to eat well, play basketball and spend time with my family.
